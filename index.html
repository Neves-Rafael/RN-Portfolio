Skills

HTML

HTML, ou Hypertext Markup Language, é a linguagem base da web. Com suas tags, permite estruturar e formatar conteúdo, como texto, imagens e links, criando páginas interativas. Junto com CSS e JavaScript, forma o trio essencial para desenvolver sites e aplicativos web.

CSS

CSS (Cascading Style Sheets) é uma linguagem essencial para a web que controla o design e a aparência de páginas HTML, permitindo ajustar cores, fontes, layout e elementos visuais para criar sites atraentes e consistentes.

JavaScript

JavaScript é uma linguagem de programação amplamente usada na web. Ela permite tornar páginas da web interativas, adicionando funcionalidades dinâmicas como animações, validação de formulários e atualizações em tempo real. Compatível com a maioria dos navegadores, o JavaScript é crucial para o desenvolvimento web moderno.

(Estudando)

TailwindCSS

Tailwind CSS é um framework front-end que agiliza o desenvolvimento web. Usa classes CSS predefinidas para criar interfaces responsivas e estilizadas, eliminando a necessidade de escrever CSS personalizado. Altamente configurável, é uma escolha popular para criar designs eficientes e personalizados.

REACT

React é uma biblioteca JavaScript para criar interfaces de usuário interativas. Usa componentes reutilizáveis e renderização virtual para melhorar a eficiência, tornando-o ideal para construir aplicativos web complexos e responsivos. É amplamente adotado na comunidade de desenvolvimento web

(Estudando)

Figma

Figma é uma ferramenta de design e prototipagem de interfaces de usuário baseada na web, facilitando a colaboração e edição em tempo real para equipes de design. Ideal para projetos digitais.
